### 1.Bad smell Code 怎麼來?
    廢話~不就工程師自己寫的??
    
#### 壞循環1 
    1.bad code 產 Bug 。
    2.解 Bug 花時間，時間都花在這。
    3.沒了時間只好求能動就好，所以寫出更多的 Bad Code

    
#### 壞循環2
    1.Bad Code 又臭又長改不動
    2.新功能想加只能想辦法塞，造成....自己想像。

![Bad Code Loop](https://i.imgur.com/S3J5Lxb.jpg)

    
### 2.為什麼要解 Bad Code
    1.看到凌亂的桌子不會想整理嗎?
    2.擺滿了東西怎麼在放新的東西呢?
    3.結論，如果你還想繼續用你的桌子就整理，就這麼簡單。

### 4.Code Smell 的特徵

    2.Duplicated Code
    3.Feature Envy

    5.Long Method
    6.Primitive Obsession

### 5.Refectoring 的手法
